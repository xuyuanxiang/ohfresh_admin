ohFresh|探鲜家 卖家版
==========

ohFresh|探鲜家卖家版前端工程。

## 依赖

  * NodeJS (http://nodejs.org/)
  * Grunt (http://gruntjs.com/)
  * Bower (http://bower.io/)

## 运行

  * 克隆或者下载项目
  ```
  $ git clone https://github.com/xuyuanxiang/ohfresh_admin.git
  ```
  * 安装项目依赖
  ```
  $ npm install
  $ bower install
  ```
  * 构建项目
  ```
  $ grunt build
  ```
  * 启动项目（http://localhost:3100/）
  ```
  $ grunt server
  ```

## 发布

  * 将项目打包发布到dist/目录下
  ```
  $ grunt publish
  ```
